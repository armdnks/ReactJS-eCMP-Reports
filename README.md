# React.js - eCMP Reports Frontend

> React.js, Node.js and MySQL

## Usage

### Install npm packages

```bash
npm install
```

> `optional` - if there is any issue with the packages

```bash
npm install --legact-peer-deps
```

```bash
npm audit fix --force
```

### Run react app

```bash
npm start
```
